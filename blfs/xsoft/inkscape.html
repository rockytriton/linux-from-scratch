Inkscape-1.0

Introduction to Inkscape

Inkscape is a what you see is what you get Scalable Vector Graphics editor. It is useful for creating, viewing and changing SVG images.

This package is known to build and work properly using an LFS-10.0 platform.

Package Information

  • Download (HTTP): https://media.inkscape.org/dl/resources/file/inkscape-1.0.tar.xz

  • Download MD5 sum: e5f1ee6b32ac0a94bdd5d99190e7bb9e

  • Download size: 30 MB

  • Estimated disk space required: 722 MB (165 MB installed)

  • Estimated build time: 5.7 SBU (with parallelism=4)

Note

The tarball inkscape-1.0.tar.xz will extract to inkscape-1.0_2020-05-01_4035a4fb49

Inkscape Dependencies

Required

Boost-1.74.0, double-conversion-3.1.5, GC-8.0.4, GDL-3.34.0, Gsl-2.6, Gtkmm-3.24.2, libsoup-2.70.0, libxslt-1.1.34, Poppler-20.08.0, popt-1.18 and Wget-1.20.3 (to download the test dependencies)

Recommended

ImageMagick-7.0.10-27, Little CMS-2.11 or Little CMS-1.19, libcanberra-0.30 (to eliminate plugin warnings), Potrace-1.16 (for the bucket-fill tool), lxml-4.5.2 and Scour-0.37 (both at runtime, for Save As Optimized SVG).

Optional

Aspell-0.60.8, dbus-1.12.20 (to run inkscape from scripts), Doxygen-1.8.19, gtkspell, libcdr, libvisio, libwpg (or libwpd) and NumPy (at runtime for some extensions)

Optional Runtime Dependencies (for some of the Inkscape extensions)

XML::XQL

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/inkscape

Installation of Inkscape

First, fix build failure with glibc-2.32: 

sed -i '/#include <iostream>/a #include <atomic>' src/ui/tool/node.cpp

Install Inkscape by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr \
      -DCMAKE_BUILD_TYPE=Release  \
      ..                          &&
make

This package does not come with a working test suite.

Now, as the root user: 

make install

Note

This package installs icon files into the /usr/share/icons/hicolor hierarchy and desktop files into the /usr/share/applications hierarchy. You can improve system performance and memory usage by updating /usr/share/icons/hicolor/index.theme and /usr/share/applications/mimeinfo.cache. To perform the update you must have GTK+-2.24.32 or GTK+-3.24.22 installed (for the icon cache) and desktop-file-utils-0.26 (for the desktop cache) and issue the following commands as the root user: 

gtk-update-icon-cache -qtf /usr/share/icons/hicolor &&
update-desktop-database -q

Command Explanations

-DCMAKE_BUILD_TYPE=Release: This switch is used to build the release library without any debug `assert` in the code.

-DWITH_DBUS=ON: use this if you wish to use inkscape in interactive scripts which manipulate images.

Contents

Installed Programs: inkscape and inkview
Installed Library: libinkscape_base.so
Installed Directories: /usr/lib/inkscape and /usr/share/inkscape

Short Descriptions

inkscape

is a SVG (Scalable Vector Graphics) editing program.

inkview

is a simple program for displaying SVG files.

libinkscape_base.so

provides the routines used by inkscape and inkview.
