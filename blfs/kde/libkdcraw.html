libkdcraw-20.08.0

Introduction to libkdcraw

Libkdcraw is a KDE wrapper around the libraw-0.20.0 library for manipulating image metadata.

This package is known to build and work properly using an LFS-10.0 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/release-service/20.08.0/src/libkdcraw-20.08.0.tar.xz

  • Download MD5 sum: 06c454b85bf4663859936dcf19044f2b

  • Download size: 44 KB

  • Estimated disk space required: 1.3 MB

  • Estimated build time: less than 0.1 SBU

libkdcraw Dependencies

Required

KDE Frameworks-5.73.0 and libraw-0.20.0

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/libkdcraw

Installation of libkdcraw

Install libkdcraw by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Programs: None
Installed Library: libKF5KDcraw.so
Installed Directories: $KF5_PREFIX/include/KF5/KDCRAW and $KF5_PREFIX/lib/cmake/KF5KDcraw
