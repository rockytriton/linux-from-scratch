Plasma-wayland-protocols-1.1.1

Introduction to the Plasma-wayland-protocols

This package provides a custom set of protocol definitions for KDE.

This package is known to build and work properly using an LFS-10.0 platform.

Package Information

  • Download (HTTP): https://download.kde.org/stable/plasma-wayland-protocols/plasma-wayland-protocols-1.1.1.tar.xz

  • Download MD5 sum: 82207f4df5e4a91452339bcf7b983d7e

  • Download size: 38 KB

  • Estimated disk space required: 680 KB

  • Estimated build time: less than 0.1 SBU

Plasma-wayland-protocols Dependencies

Required

extra-cmake-modules-5.73.0

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/plasma-wayland-protocols

Installation of Plasma-wayland-protocols

Install Plasma-wayland-protocols by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=/usr ..

Now, as the root user: 

make install

Contents

Installed Programs: none
Installed Libraries: none
Installed Directories: $KF5_PREFIX/share/plasma-wayland-protocols and $KF5_PREFIX/lib/cmake/PlasmaWaylandProtocols
